# Clearcut

## Another image background removal app.

### Getting started

```bash
git clone https://github.com/Eppu/clearcut.git
cd clearcut
npm install
npm run dev
```

Built with

- React
- Vite
- TailwindCSS
- [shadcdn/ui](https://ui.shadcn.com/)
- [imgly/background-removal-js](https://github.com/imgly/background-removal-js)
